# PocketSeoul
### URL : http://pocketseoul.duckdns.org:8080/seoul/
#### 개발기간 : 23.01.14 ~ 
산발적으로 흩어진 정보들을 모아 확인하고 관리할 수 있는 서울 청년들을 위한 사이트입니다.

## 개발 환경
- STS 3
- JDK 8
- Oracle Cloud
- ubuntu 22.04
- Tomcat 8.5.83
- Spring 5.0.2
- SQL Developer
